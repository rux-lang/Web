# Process and Thread

Process termination, suspension, and identifiers.

**Module:** `Windows`

| Function                                         | Description                         |
| ------------------------------------------------ | ----------------------------------- |
| [`ExitProcess`](exitprocess)                     | Terminate the process.              |
| [`GetCurrentProcessId`](getcurrentprocessid)     | Return the calling process ID.      |
| [`GetCurrentThreadId`](getcurrentthreadid)       | Return the calling thread ID.       |
| [`Sleep`](sleep)                                 | Suspend the current thread.         |
