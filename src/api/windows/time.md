# Time

Monotonic tick count and broken-down wall-clock time.

**Module:** `Windows`

| Function                             | Description                              |
| ------------------------------------ | ---------------------------------------- |
| [`GetLocalTime`](getlocaltime)       | Retrieve current local time.             |
| [`GetSystemTime`](getsystemtime)     | Retrieve current UTC time.               |
| [`GetTickCount64`](gettickcount64)   | Milliseconds elapsed since startup.      |

See [`SystemTime`](types#systemtime) for the wall-clock result structure.
