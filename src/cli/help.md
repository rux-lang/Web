# `rux help`

Show help information for the CLI.

```sh
rux help
rux -h
rux --help
```

Show help information for a specific command.

```sh
rux help [command]
```

When used as a global option, `-h` and `--help` print the help screen and exit without running a command.

## Examples

```sh
rux help
rux help build
rux help run
```
