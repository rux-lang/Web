# `rux clean`

Remove all build artifacts and temporary files.

```sh
rux clean [options]
```

Removes `Bin/` and `Temp/` folders.

## Options

| Option   | Description                    |
| -------- | ------------------------------ |
| `--temp` | Removes only `Temp/` directory |

## Examples

```sh
rux clean
rux clean --temp
```
