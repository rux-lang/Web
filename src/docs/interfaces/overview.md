# Interfaces

An interface declares a set of method signatures that a type must satisfy, enabling shared behavior across unrelated types. Each aspect has a dedicated chapter:

| Topic                                                  | Description                                       |
| ------------------------------------------------------ | ------------------------------------------------- |
| [Declaration](/docs/interfaces/declaration)            | Declaring the methods an interface requires       |
| [Implementation](/docs/interfaces/implementation)      | Satisfying an interface with `extend`             |
