# Directory Layout

```
MyPackage/
├── Rux.toml          # Package manifest
├── Src/
│   └── Main.rux      # Source files (bin) or Lib.rux (lib)
├── Bin/
│   ├── Debug/        # Debug build output
│   └── Release/      # Release build output
└── Temp/             # Intermediate build files
```

`Bin/` and `Temp/` are generated by the toolchain and should be listed in `.gitignore`.
