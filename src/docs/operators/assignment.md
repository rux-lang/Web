# Assignment Operators

| Operator | Equivalent to    |
| -------- | ---------------- |
| `=`      | Plain assignment |
| `+=`     | `x = x + rhs`    |
| `-=`     | `x = x - rhs`    |
| `*=`     | `x = x * rhs`    |
| `/=`     | `x = x / rhs`    |
| `%=`     | `x = x % rhs`    |
| `&=`     | `x = x & rhs`    |
| `\|=`    | `x = x \| rhs`   |
| `^=`     | `x = x ^ rhs`    |
| `<<=`    | `x = x << rhs`   |
| `>>=`    | `x = x >> rhs`   |
