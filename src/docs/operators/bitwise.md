# Bitwise Operators

| Operator | Operation           |
| -------- | ------------------- |
| `&`      | Bitwise AND         |
| `\|`     | Bitwise OR          |
| `^`      | Bitwise XOR         |
| `~`      | Bitwise NOT (unary) |
| `<<`     | Left shift          |
| `>>`     | Right shift         |
