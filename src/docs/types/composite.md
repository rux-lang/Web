# Composite Types

| Construct       | Description                                        |
| --------------- | -------------------------------------------------- |
| `struct`        | A named collection of fields                       |
| `enum`          | A type with a fixed set of variants                |
| `union`         | An untagged union of types                         |
| `T[N]`          | A fixed-size array of `N` elements of type `T`     |
| `[T]`           | A slice — a dynamically-sized view into a sequence |
| `(T1, T2, ...)` | A tuple                                            |
