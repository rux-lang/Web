# Functions

A function groups a sequence of statements under a name so it can be invoked repeatedly. Rux supports several kinds of functions, each covered in a dedicated chapter:

| Kind                                       | Description                                          |
| ------------------------------------------ | --------------------------------------------------- |
| [Declaration](/docs/functions/declaration) | Declaring functions, parameters, and return types   |
| [Variadic](/docs/functions/variadic)       | Functions that accept any number of arguments       |
| [Generic](/docs/functions/generic)         | Functions parameterized over types                  |
| [Assembler](/docs/functions/assembler)     | Functions whose body is written in assembly         |
| [Main](/docs/functions/main)               | The program entry point                             |
